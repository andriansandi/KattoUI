# AI Instructions for KattoUI

This directory contains rules for AI coding agents working on KattoUI.

Start with `session-start.md` every session. Then consult the relevant guides before making changes.

- `architecture.md` — monorepo boundaries and where code belongs.
- `coding-standards.md` — general code quality and UX rules.
- `typescript.md` — TypeScript conventions.
- `cloudflare.md` — Worker and Cloudflare-specific rules.
- `design.md` — visual and accessibility standards.
- `testing.md` — testing strategy.
- `security.md` — security checklist.
- `documentation.md` — when and how to update docs.
- `commit-convention.md` — commit message format.
- `review-checklist.md` — final checklist before finishing.
