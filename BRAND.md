# KattoUI Brand Guidelines

This document defines the identity of KattoUI. It is the source of truth for everyone designing, writing, or building KattoUI. It is not a marketing document. It is not a logo guideline.

---

## Mission

KattoUI exists to make working with AI feel simple, beautiful, and reliable. Developers should spend their time building with AI, not configuring AI tools.

---

## Vision

Become the most elegant open-source AI workspace. Not the biggest. Not the most feature rich. The most enjoyable to use.

---

## Positioning

| KattoUI is | KattoUI is not |
|------------|----------------|
| Open source | A ChatGPT clone |
| Cloudflare first | An Open WebUI clone |
| Developer focused | A generic admin dashboard |
| Self hostable | A toy project |
| Extensible | |
| Professional | |

---

## Brand Personality

Imagine KattoUI as a person.

**Is:**
- Friendly
- Calm
- Professional
- Curious
- Helpful
- Reliable
- Fast
- Confident

**Never:**
- Loud
- Childish
- Arrogant

---

## The Cat

The cat is inspiration, not a mascot. Cats represent curiosity, focus, independence, comfort, and quiet confidence. The UI should communicate these values. Do not fill the interface with cat graphics.

---

## Design Keywords

Use these words when making design decisions:

- Warm
- Minimal
- Thoughtful
- Calm
- Fast
- Focused
- Elegant
- Developer
- Modern
- Timeless

See [DESIGN.md](./DESIGN.md) for the visual language.

---

## Avoid

- Busy
- Playful
- Noisy
- Corporate
- Cold
- Overdesigned
- Overanimated
- Cute
- Cartoon

---

## Voice

Write like an experienced developer. Short sentences. Clear language. No hype. No buzzwords. No unnecessary adjectives. Explain only what matters.

---

## Tone

**Is:**
- Helpful
- Honest
- Direct
- Respectful
- Professional

**Never:**
- Sarcastic
- Overly casual
- Exaggerated

---

## Writing Style

| Prefer | Instead of |
|--------|------------|
| Start Chat | Let's get started! |
| Connect Provider | Awesome! |
| Choose Model | Amazing! |
| Theme | Super powerful! |
| Workspace | Magic AI! |

---

## UI Personality

Every screen should feel:

- Fast
- Clean
- Quiet
- Intentional

Never crowded. Never overwhelming.

---

## Color Philosophy

Neutral colors dominate. Orange creates focus. Orange should feel earned, not everywhere.

---

## Motion Philosophy

Motion exists to explain, not to impress. Every animation should communicate change, feedback, state, or hierarchy. Nothing should move without purpose.

---

## Product Principles

Chat is the center. Everything supports chat: providers, models, themes, plugins, agents, and the marketplace. See [ROADMAP.md](./ROADMAP.md) for the build sequence.

---

## User Experience

Users should feel:
- "I understand this immediately."
- "I know where everything is."
- "I trust this."
- "I want to keep using this."

---

## Community

Contributors should value:

- Consistency
- Quality
- Simplicity
- Accessibility
- Performance
- Maintainability

A smaller polished feature is always better than a larger unfinished one.

---

## One Sentence

> KattoUI is the professional AI workspace with the quiet confidence of a cat.
